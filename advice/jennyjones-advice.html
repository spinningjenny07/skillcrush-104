Hi my name is Jenny and I have some advice for you!

Don't Give Up

There is LOT to learn when you're first starting to code. Details, details, details.
There will be days when you feel like you'll never understand or be able to do anything real with your new skills.
Don't give into discouragement.
It will eventually make sense and then you can do some really cool stuff with code!

Pick Something and Stick with It! (AKA: Fight Shiny Object Syndrome!)

There are SO many languages, SO many technologies, SO many ways you can go on your coding journey.
So, when you first start out, pick one thing and learn it and then move on to something else.
Don't jump around to different languages when you first start.
Pick one and learn it well because then, you can transfer what you learn to any other language.

Find Support

Don't Code Alone!
Find friends you can code with! Go to a meetup and get some help.
It's a lot more fun to code when you can share your work with others and have the chance to get ideas from other people's work as well!
